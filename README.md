# WSO2 EI Integration Examples

This repository contains applied examples of integration development using the WSO2 Enterprise Integrator (WSO2 EI).

## Project List

1. **CacheExampleProject** - Demonstrates caching mechanisms.
2. **CloneAndAggregateExample** - Shows cloning and aggregating messages.
3. **CloneAndAggregateOfficialExample** - Official example for cloning and aggregating.
4. **DBLookupExampleProject** - Example of database lookup operations.
5. **DBReportExample** - Demonstrates reporting using database data.
6. **FilterMediatorProjectExample** - Filtering messages based on conditions.
7. **HeaderBasedRoutingExampleProject** - Routing messages based on headers.
8. **HeaderBasedRoutingOfficialExampleProject** - Official header-based routing example.
9. **HelloWorldProject** - Simple Hello World project.
10. **HttpResponseCodeExampleProject** - Handling HTTP response codes.
11. **IterateAndAggregateExample** - Iterating and aggregating messages.
12. **JsonPlaceHolderIntegration** - Integration with JSONPlaceholder.
13. **JsonToXMLOfficialExampleProject** - Official JSON to XML transformation example.
14. **JsonToXmlExampleProject** - JSON to XML transformation.
15. **MessageStoreExampleProject** - Storing and retrieving messages.
16. **PayLoadFactoryExampleProject** - Using PayloadFactory for message transformation.
17. **ProxyServiceExampleProject** - Creating and using proxy services.
18. **SOAPExampleOfficialDoc** - Official SOAP example documentation.
19. **SOAP_PostmanAPIs_To_RESTApis** - Converting SOAP APIs to REST APIs using Postman.
20. **ScheduleTaskExampleProject** - Scheduling tasks.
21. **SequenceTemplateProjectExample** - Using sequence templates.
22. **UsingPropertiesExampleProject** - Utilizing properties in integrations.
23. **ValidateJsonRequestExampleProject** - Validating JSON requests.

Each project folder contains detailed information and instructions on how to set up and run the examples.

## Getting Started

To get started with any of the examples, navigate to the respective project folder and follow the instructions in the `README.md` file.
